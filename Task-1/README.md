**Task 1 – Mobile App Signup Flow (LearnEveryDay)**

_Description_

Designed a modern and user-friendly mobile app signup flow for LearnEveryDay, an online learning platform. The objective was to create a simple and engaging onboarding experience that allows users to register, log in, and start learning with minimal effort.

_Features_

Splash / Welcome Screen
Create Account Screen
Login Screen
Account Creation Success Screen
Clean and minimal UI
Consistent color palette and typography
User-friendly form layout with clear navigation

_Design Tools_

Figma
Design Principles Used
Simplicity and consistency
Visual hierarchy
Accessibility
Easy navigation
User-centered design

_Outcome_

This project demonstrates my understanding of mobile onboarding flows, UI consistency, spacing, typography, and creating intuitive user experiences.
