**Task 3 – Restaurant Menu UI**

_Description_

Designed a modern restaurant menu interface for Delicious Bites. The menu focuses on improving customer experience by presenting food categories, pricing, and images in a clean, organized, and visually attractive layout.

_Features_

Sidebar navigation
Food categories
Search bar
High-quality food images
Item cards with prices
Add-to-cart buttons
Restaurant information
Contact and delivery section
Clean and organized layout

_Design Tools_

Figma

_Design Principles Used_

Easy navigation
Clear categorization
Visual consistency
Balanced spacing
User-friendly interface

_Outcome_

This project improved my skills in designing information-rich interfaces while maintaining usability, readability, and aesthetic appeal.
