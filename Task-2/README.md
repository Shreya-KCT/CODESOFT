**Task 2 – Email Template Design**

_Description_

Designed a responsive promotional email template for LearnEveryDay. The template is intended to encourage users to continue learning by highlighting new courses, weekly updates, and featured content while maintaining a clean and visually appealing layout.

_Features_

Professional newsletter layout
Hero section with call-to-action
Featured learning categories
Continue Learning button
Social media section
Footer with unsubscribe and privacy links
Clean typography and balanced spacing

_Design Tools_

Figma

_Design Principles Used_

Visual hierarchy
Readability
Responsive layout
Consistent branding
Effective use of whitespace

_Outcome_

This project helped me understand email layout structure, content organization, call-to-action placement, and designing templates suitable for marketing and educational platforms.
